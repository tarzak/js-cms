# js-cms
Modular JavaScript CMS based on io.js &amp; backbone.js

## Technologies:
- ES6
- io.js, koa.js
- backbonejs + react
- MongoDB

## Basic features:
- User registration
- User logining
- Roles: admin, registered user
- Adding posts
- Editing posts
- Removing posts
 
## DB structure:
- Posts collection
- Roles collection
- Users collection
