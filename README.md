# js-cms
Modular JavaScript CMS based on io.js &amp; backbone.js

## Technologies:
- ES6
- io.js, koa.js
- backbonejs + react
- MongoDB

## Basic features:
- User registration
- User logining
- Roles: admin, registered user
- Adding posts
- Editing posts
- Removing posts

## Modules:
- Log
- Debug
- Config
- ? Statistic module (to gather various information)
 
## MongoDB collections structure:
- Posts
- Roles
- Users
